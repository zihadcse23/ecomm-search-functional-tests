# E-Commerce Search Functionality Test
Manual QA Test Cases for Search Feature. Website under Test: https://demo.evershop.io

### Overview
This project contains a comprehensive set of manual test cases for validating the Search functionality of the Demo EverShop e-commerce website. The test suite covers UI validation, functional testing, negative testing, scalability, multi-keyword search, fuzzy search, multilingual behavior, analytics behavior, and more.

This project was created as part of a professional QA portfolio and follows industry-standard test case design practices.

### Objectives
- Validate end-to-end Search flow.
- Ensure best-practice coverage for e-commerce search.
- Identify usability issues.
- Report functional defects with proper bug reports.
- Evaluate performance and scalability of search results.
